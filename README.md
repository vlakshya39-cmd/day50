# Defend Your Experience

An AI-powered interview preparation application that transforms resumes into adaptive mock interviews.

## Overview

Defend Your Experience focuses on validating interview readiness instead of improving resume writing. Every statement in a resume becomes a claim that the AI interviewer challenges with progressively deeper questions.

The application evaluates how confidently users defend their experience and identifies areas that require better preparation.

## Features

- Resume upload and text paste
- Automatic claim extraction
- AI-powered adaptive interview
- Confidence tracking
- Personalized defense report
- Export interview results
- Responsive modern interface

## Tech Stack

- CSS
- JavaScript
- Claude API
