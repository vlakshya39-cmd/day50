# Day 50 - Defend Your Experience

## Project Overview

Defend Your Experience is an AI-powered interview preparation application that helps users validate the claims made on their resume or portfolio.

Instead of reviewing resume formatting, the application conducts an adaptive interview where every claim is challenged with follow-up questions. The goal is to improve interview confidence by identifying strong explanations and weak areas before a real interview.

## Features

- Upload or paste resume content
- AI extracts resume claims automatically
- Adaptive interview based on previous answers
- Dynamic confidence tracking
- Interactive interview interface
- Session management
- Export interview data
- Modern responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Claude AI API
